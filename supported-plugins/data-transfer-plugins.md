AdvancedTeleport allows the user to import and export data between itself and other plugins, making transferring for those switching from other plugins easy.&#x20;

## Essentials

The modern version of Essentials - by the EssentialsX team - is a plugin that allows data transfer between itself and AT. To import data from Essentials to AT, do `/at import essentials`, or to export data to Essentials from AT, do `/at export essentials`.

The following data can be exchanged between AT and Essentials:
